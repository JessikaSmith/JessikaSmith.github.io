# Вопросы

*Вопрос 1.1:* Посмотрев на страницу, содержащую таблицу, каковы различия между DOM, как показано инспектором DOM и исходным кодом HTML? Зачем использовать инспектор DOM? Когда источник HTML полезен?  

*Ответ:* Инспектор DOM отображает не только содержимое HTML, но и позволяет вносить изменение в структуру страницы. Источник HTML полезен, если страница не содержит динамических элементов.   

*Вопрос 1.2:* Ниже мы частично воспроизвели первые строки из набора данных таблицы. Какая часть программного обеспечения генерирует эту таблицу? Где хранятся исходные данные?  

*Ответ:* Таблица подгружается с сервера при запуске страницы в браузере и отрисовывается согласно инструкциям html, css и js файлов.     

*Вопрос 2.1:* Вы создали фильтры по континентами, которые представляют собой ограниченный набор категориальных данных. Можно ли фильтровать другие столбцы из таблицы таким же образом? Например вы бы использовали чекбоксы или другой HTML-виджет?  

*Ответ:* Такие столбцы, как, например, количество населения или ожидаемое количество лет уместнее фильтровать с помощью ползунка, когда будет происходить отображение табличных значений по признаку *value >= выставленное значение*.

*Вопрос 5.1:* Каковы плюсы и минусы использования HTML и SVG? Приведите примеры в контексте создания визуализации.   

*Ответ:* SVG - векторный формат, что позволяет легко манипулировать размером визуализации не ухудшая качество. При этом, возможно динамическое изменение изображения, в отличии от html, где просто происходит перерисовка. Ограничения - большой объем данных может вызывать сильное замедление при работе с SVG.

*Вопрос 7.1:* Приведите пример ситуации, в которых подходит визуализация, следуя аргументам, обсуждаемым в лекции и в учебнике (пример не может быть таким же, как упоминается в лекции или учебнике).  

*Ответ:*  Для отображения рельефа подходит метод изолиний (линий одного уровня), которые при помощи цвета кодируют высоту. Подобные карты являются общепринятым способом изображения местности.

*Вопрос 7.2:* Какие ограничения статических диаграмм вы можете решить с помощью интерактивности?

*Ответ:* Интерактивность помогает проследить динамику изменений, например, можно плавными переходами гистограммы отображать годовые переходы значений.  

*Вопрос 7.3:* Каковы ограничения визуализации?

*Ответ:* Не всегда можно отобразить полную и точную информацию о каком-либо комплексном объекте и тогда приходится делать выбор о том, что является наиболее важным.  

*Вопрос 7.4:* Почему семантика важна для данных?  

*Ответ:* Семантика вводит внутренние связи данных, что облегчает процесс обработки.
